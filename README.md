Соревнование: https://www.kaggle.com/competitions/stock-price-prediction-challenge/overview

Задача: предсказать цену пяти безымянных акций на 10 дней вперед.

В качестве моделей использовались: 

1) AutoTS из LightAutoML: признак таргета, временные + lag признаки;
2) Реализация LSTM на pytorch: признак таргета, временные, lag признаки, дополнительные исторические признаки, процентные соотношения изменения по разным признакам.

Основной вывод из работы: несмотря на то, что LSTM кажется более сильным методом, и в примере решения соревнования LSTM, обучаемая на одном признаке, выигрывает у AutoTS, при добавлении новых фичей она легко переобучается и проигрывает AutoTS, поэтому к выбору фичей для задач, связанных с предиктом цены на акции, нужно подходить очень аккуратно.
